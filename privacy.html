Política de Privacidad de Petfy

Última actualización: 10 de junio de 2026

```

Esta Política de Privacidad explica cómo Petfy recopila, usa, almacena y protege la información relacionada con el uso de la aplicación Petfy: Cuidado de Mascotas.

Petfy es una aplicación diseñada para ayudar a los usuarios a organizar el cuidado de sus mascotas mediante perfiles, recordatorios, registros de salud, alimentación, peso, notas, fotografías y funciones de asistencia general.

Petfy no reemplaza la atención veterinaria profesional, no ofrece diagnóstico veterinario, no prescribe tratamientos y no está diseñada para emergencias veterinarias.

1. Responsable de la aplicación

La aplicación Petfy es operada por Arvilo Labs, con base en Nicaragua.

Para consultas sobre privacidad, puedes escribir a: support@petfy.app

2. Información que podemos recopilar

Petfy puede recopilar o almacenar los siguientes tipos de información, según las funciones que el usuario utilice:

  • Datos de cuenta, como correo electrónico e identificador de usuario.
  • Información de perfil del usuario, si el usuario la agrega.
  • Datos de mascotas, como nombre, especie, raza, fecha de nacimiento, sexo, peso y notas.
  • Registros de salud de mascotas, como vacunas, medicamentos, desparasitación, controles, peso y observaciones.
  • Recordatorios creados por el usuario.
  • Registros de alimentación y cuidado.
  • Fotos de mascotas, si el usuario decide agregarlas.
  • Consultas o mensajes enviados a Petfy AI.
  • Información de compras o suscripciones gestionadas mediante Google Play o servicios de suscripción.
  • Información técnica básica necesaria para seguridad, diagnóstico de errores y funcionamiento de la app.
  • Ubicación aproximada o precisa, si el usuario utiliza funciones relacionadas con veterinarias cercanas o mapas.

3. Cómo usamos la información

Usamos la información para:

  • Crear y administrar la cuenta del usuario.
  • Guardar y mostrar perfiles de mascotas.
  • Gestionar recordatorios, registros de salud, alimentación y peso.
  • Generar reportes o documentos relacionados con las mascotas, si el usuario lo solicita.
  • Permitir el funcionamiento de funciones Premium o Premium Plus.
  • Procesar o validar suscripciones mediante Google Play o proveedores de suscripción.
  • Ofrecer asistencia general mediante Petfy AI.
  • Mejorar la estabilidad, seguridad y funcionamiento de la aplicación.
  • Ayudar a encontrar veterinarias cercanas, si esa función está disponible y el usuario la utiliza.

4. Petfy AI

Petfy AI puede procesar preguntas, mensajes o contexto relacionado con mascotas que el usuario decida enviar. Esta función ofrece orientación general sobre cuidado de mascotas.

Petfy AI no proporciona diagnóstico veterinario, no reemplaza a un veterinario, no prescribe medicamentos y no debe usarse para emergencias. Ante cualquier problema médico o situación urgente, el usuario debe consultar a un profesional veterinario.

5. Fotos y archivos

Si el usuario agrega fotos de mascotas u otros archivos, estos pueden almacenarse para mostrarse dentro de la cuenta del usuario y para las funciones relacionadas de Petfy.

6. Ubicación

Petfy puede solicitar o usar ubicación si el usuario utiliza funciones de búsqueda de veterinarias cercanas o mapas. Petfy no comparte la ubicación precisa actual del usuario con otros usuarios.

7. Compras y suscripciones

Petfy puede ofrecer funciones Premium o Premium Plus mediante compras dentro de la aplicación o suscripciones. Estas compras pueden ser procesadas por Google Play y/o proveedores de gestión de suscripciones.

Petfy no almacena directamente los datos completos de tarjetas de pago del usuario.

8. Proveedores de servicios

Para operar la aplicación, Petfy puede usar proveedores externos, incluyendo servicios de autenticación, base de datos, almacenamiento, suscripciones, mapas, análisis de errores y asistencia de IA. Estos proveedores pueden procesar información únicamente para permitir el funcionamiento de la app.

Entre los proveedores que pueden utilizarse se incluyen:

  • Supabase, para autenticación, base de datos y almacenamiento.
  • Google Play, para compras y suscripciones.
  • RevenueCat, para gestión de suscripciones, si está habilitado.
  • Sentry, para monitoreo de errores y estabilidad.
  • Google Maps o Google Places, para funciones de ubicación y veterinarias cercanas, si están habilitadas.
  • Servicios de inteligencia artificial, para procesar consultas de Petfy AI, si el usuario utiliza esa función.

9. Compartición de información

Petfy no vende datos personales del usuario. Podemos compartir información solo cuando sea necesario para operar la aplicación, cumplir obligaciones legales, proteger la seguridad del servicio o procesar funciones solicitadas por el usuario.

10. Conservación de datos

Conservamos los datos mientras la cuenta del usuario esté activa o mientras sea necesario para ofrecer el servicio. El usuario puede solicitar la eliminación de su cuenta y datos asociados.

11. Seguridad

Aplicamos medidas razonables para proteger la información del usuario, incluyendo autenticación, controles de acceso y almacenamiento seguro mediante proveedores especializados. Sin embargo, ningún sistema es completamente infalible.

12. Derechos del usuario

El usuario puede solicitar:

  • Acceso a sus datos.
  • Corrección de información.
  • Eliminación de cuenta y datos asociados.
  • Información sobre el uso de sus datos.

13. Eliminación de cuenta y datos

El usuario puede solicitar la eliminación de su cuenta y datos asociados desde la aplicación, si la función está disponible, o escribiendo a support@petfy.app.

También puedes consultar la página de eliminación de cuenta y datos: Eliminar cuenta y datos.

14. Menores de edad

Petfy no está dirigida específicamente a niños. La aplicación está orientada a personas responsables del cuidado de mascotas.

15. Cambios a esta política

Podemos actualizar esta Política de Privacidad cuando sea necesario. La versión actualizada estará disponible en esta misma página.

16. Contacto

Para preguntas sobre esta Política de Privacidad, escribe a: support@petfy.app
```
